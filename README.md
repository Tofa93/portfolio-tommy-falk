# Portfolio Website - Tommy Falk

Persönliche Portfolio-Website von Tommy Falk. Die Seite stellt Projekte, Kenntnisse, Lebenslauf und Kontaktmöglichkeiten in einem modernen Dark-Design mit Neon-Akzenten vor.

## Inhalte

- Persönlicher Hero-Bereich mit animiertem Canvas-Hintergrund
- Kurzprofil mit Ausbildung, Frontend-Fokus und .NET/WPF-Kenntnissen
- Projektübersicht mit Filterfunktion
- Tech-Stack-Bereich
- Lebenslauf-Sektion mit Portrait
- Kontaktformular mit Mailto-Funktion
- Dark Mode mit optionalem Light Mode
- Responsive Layout für Desktop, Tablet und Smartphone

## Tech Stack

- HTML
- CSS
- JavaScript
- Canvas API
- GitHub Pages geeignet

## Lokal starten

Das Projekt benötigt kein Build-Tool.

1. Repository herunterladen oder klonen
2. Ordner in Visual Studio Code öffnen
3. `index.html` im Browser öffnen

Optional mit der VS-Code-Erweiterung **Live Server**:

```text
Rechtsklick auf index.html -> Open with Live Server
```

## Anpassungen

Die wichtigsten Inhalte können direkt in diesen Dateien geändert werden:

- `index.html` für Texte und Seitenstruktur
- `styles.css` für Farben, Layout und Design
- `script.js` für Projekte, Filter, Theme-Wechsel und Animation
